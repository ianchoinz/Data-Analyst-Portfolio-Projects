# Power BI Portfolio

Data visualisation and analytics reports completed as part of the **Coursera Microsoft Power BI Data Analyst** programme, covering ETL, data modelling, DAX, and report design.

---

## Reports

### 01 · Tailwind Traders Sales Dashboard
**`01_tailwind_traders/`**

Multi-page dashboard analysing Tailwind Traders sales performance across five countries (UK, USA, Australia, France, UAE). Includes KPIs for loyalty points, product sales ranking, YTD profit margin, and net revenue by product. DAX measures used for time intelligence and profit margin calculations.

**Skills:** DAX · Profit Margin Analysis · Time Intelligence · Multi-page Report Design

| File | Description |
|------|-------------|
| `Tailwind_Traders_Report.pdf` | Completed Power BI report |
| `Tailwind-Traders-Sales.xlsx` | Source data |

---

### 02 · AdventureWorks Sales Analysis
**`02_performing_analysis/`**

Analysis of 295 products and 109M+ in revenue using funnel charts, customer age group histograms, and city-level scatter plots. Demonstrates customer segmentation and geographic revenue distribution.

**Skills:** Funnel Chart · Histogram · Scatter Plot · Customer Segmentation

| File | Description |
|------|-------------|
| `Performing-an-analysis.pdf` | Completed Power BI report |
| `Adventure-Works-Data.xlsx` | Source data |

---

### 03 · Order Status Tracking Report
**`03_sales_report/`**

Interactive sales report monitoring order pipeline (Shipped/Processing/Cancelled) across three regions (Asia, Europe, North America) with monthly breakdown by product category. Utilises matrix tables and cross-page filters.

**Skills:** Matrix Table · Order Status Tracking · Regional Analysis · Drill-through

| File | Description |
|------|-------------|
| `Sales-Report.pdf` | Completed Power BI report |
| `Orders.csv` | Order transaction data |
| `Customers.csv` | Customer data |
| `Products.csv` | Product data |
